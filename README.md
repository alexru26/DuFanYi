## How to Use
Currently, there are no migration featuresk, so you will have to clear all the data of the app if you want a newer version. To do this, go to Settings -> Apps -> DuFanYi -> Storage -> Clear Data.

To actually use the app to read novels, first legally obtain a txt of the text. In this txt, the first three lines must be the title, author, and status, respectively. After that, the chapters titles must be "ç¬¬Xç«  ..." or just "ç¬¬Xç« " where X is a number. There should not be any blank lines at all. If you want to update a series already in your library, you have to unfortunately delete it first.
